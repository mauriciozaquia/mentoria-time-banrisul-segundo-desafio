# Desafio DBServer - Mentoria Banrisul
## Testes automatizados de API

*Script*  desenvolvido para o desafio de testes automatizados da mentoria do time **DBserver-Banrisul**, tem por objetivo testar e validar casos de uso para a API **viacep**

### Pré-requisitos 

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas: 
[Git]([https://git-scm.com](https://git-scm.com/)), 
[JDK 8]([https://www.oracle.com/br/java/technologies/javase/javase-jdk8-downloads.html](https://www.oracle.com/br/java/technologies/javase/javase-jdk8-downloads.html)), 
[JUnit 5]([https://junit.org/junit5/](https://junit.org/junit5/)), 

Além disto é bom ter uma IDE Java para trabalhar com o código como [IntelliJ IDEA]([https://www.jetbrains.com/pt-br/idea/](https://www.jetbrains.com/pt-br/idea/))
